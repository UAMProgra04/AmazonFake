# AmazonFake
AmazonFake es un proyecto de fines educativos para el uso de Web Services e integracion con una Web de consumo.
Tema: AMAZON

Integrantes:
	Andrey Padías, Bryan Fonseca, Bryan Sandí, Carlos Perez.

Objetivos Generales:
App web publicada
	- Manula de publicacion y configuracion (Itil-CobIT)
WCF publicado
	- Manula de publicacion y configuracion
Diseño Empresarial
	- Mater Page
	- web control
	- postBack
	- Cookies 
Seguridad Basada en Roles
	- Seguridad de la capa (programacion) y los usuarios
	- No poner logica en la capa de presentacion
Control Consumo Recursos (Cliente-Sevidor)
Responsive
Base de Datos
	- Minimo 15 tablas (usuario, perfiles, facturas, etc...)
Documentacion
	- Diseño del flujo de datos con [UML - DFD - SeudoCodigo] (cualquiera de las 3)
